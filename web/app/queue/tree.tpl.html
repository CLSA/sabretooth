The queue tree provides an overview of how the queues are populated.

Each branch of the tree represents a queue and includes the total number of participants belonging to that queue. Ranked queues are denoted by their rank (Qn) and highlighted. A participant must belong to a ranked queue in order to be assigned for interviewing.

Arrow buttons expand and collapse parts of the tree. Numbered buttons indicate how many participants belong in each queue in the tree. Clicking a queue button will view the queue providing you with a list of all participants belonging to it.

The above controls will refine which participants are shown by restricting to a questionnaire, particular viewing time, participant language or collection.
